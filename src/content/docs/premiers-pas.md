## 1) Installer et lancer

1. Téléchargez l’application depuis GitHub Releases.
2. Installez puis lancez l’application.

## 2) Créer votre contexte

Avant la saisie, préparez les éléments structurants :

- **Entité** : votre entreprise ou association (nom, identifiants, paramètres).
- **Plan comptable** : le référentiel de comptes que vous utiliserez.
- **Exercice** : la période comptable sur laquelle vous allez saisir.

## 3) Votre premier flux

1. Configurez votre entité.
2. Vérifiez/complétez le plan comptable.
3. Créez un exercice.
4. Saisissez une première opération.
5. Ajoutez le justificatif correspondant si disponible.
