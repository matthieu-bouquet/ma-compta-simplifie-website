## Comptes absents du modèle TPE (par rapport à l’association)

Le modèle TPE **ne contient pas** :

- Les classes **2** (immobilisations) et **8** (contributions volontaires en nature).
- Les comptes spécifiques associations : legs, fonds dédiés (191, 194–196), dons (754 associative), cotisations 756, etc.
- Le compte **5122** livret (ajoutez-le manuellement si besoin).

Pour une activité avec immobilisations significatives, complétez le plan depuis **Paramètres → Plan comptable** ou utilisez le modèle association si votre structure est une association.
