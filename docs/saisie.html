Saisie des opérations

Champs importants

  • Date : la date de l’opération.
  • Libellé : une description claire (ex. “Cotisation”, “Achat fournitures”).
  • Montant : montant et sens (débit/crédit selon le modèle de l’app).
  • Pièce justificative : attacher/associer le document si applicable.

Bonnes pratiques

  • Utilisez des libellés cohérents pour retrouver facilement les opérations.
  • Conservez les pièces (factures, reçus) associées, surtout en cas de contrôle.
  • Validez régulièrement votre saisie (ex. hebdomadaire) plutôt que de tout faire en fin de période.
