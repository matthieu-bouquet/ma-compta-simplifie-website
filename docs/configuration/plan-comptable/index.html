Plan comptable

Le plan comptable liste les comptes (numéro + libellé) disponibles pour la saisie.

Points clés

Section titled “Points clés”
  • Un plan comptable propre et cohérent facilite la saisie et les éditions.
  • Vous pouvez ajouter, modifier, ou supprimer des comptes selon vos besoins.

Recommandations

Section titled “Recommandations”
  • Conservez une logique stable dans les libellés (ex. “Achats fournitures”, “Cotisations”, etc.).
  • Évitez de multiplier les comptes si ce n’est pas nécessaire : cela rend la saisie plus lente.
